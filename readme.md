## Install

```bash
$ git clone https://github.com/elz-lang/elz.git && cd elz
$ cargo install
```

Make sure your `$GOPATH/bin` is one of `$PATH`

## Usage

Only for develop just now.

```bash
$ elz compile source.elz  # elz will compile it to llvm ir then print it.
```
